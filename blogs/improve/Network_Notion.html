    # 连接实验室主机区块链中间件问题

    **问题：**zcat连接数据库、中间件，连接失败。

    **排查：**防火墙、端口占用

    # 防火墙
    # 查看防火墙状态
    sudo ufw status
    1
    # 启动防火墙
    sudo ufw enable     //开启防火墙
    1
    # 重启防火墙
     sudo ufw reload      //重启ufw防火墙
    1
    # 添加防火墙允许通过的端口
    sudo ufw allow 22        //开放22端口
    1
    # 禁止外部访问机器内部，内部可以随意访问外部
    sudo ufw default deny
    1

    netstat -ap

    查看某一（**）端口，则可以结合grep命令：

    netstat -ap | grep **

    如查看**端口，也可以在终端中输入：

    lsof -i:**
